# Advanced Universal Ventures, LLC

Official website for Advanced Universal Ventures, LLC - A Maryland-based company specializing in SaaS solutions, tennis education consulting, and digital content creation.

## About

Advanced Universal Ventures, LLC is a forward-thinking company dedicated to excellence across multiple domains:

- **SaaS Solutions**: Custom mobile and web applications focused on health, fitness, and lifestyle
- **Tennis & Education**: Expert consulting services for student-athletes pursuing NCAA tennis opportunities
- **Content Creation**: Engaging social media content by Hanyu (Mason) Liu
- **Support & Privacy**: Comprehensive privacy policies and support for all applications

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Ready for Vercel, Netlify, or any Node.js hosting

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn/pnpm

### Installation

1. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

2. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
